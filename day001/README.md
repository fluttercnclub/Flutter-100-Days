# Day001

与其他语言或框架一样，我们学习 Flutter 的第一步也来写个 "Hello World"，本指南用 VSCODE 来学习编码，假定你已经按照教程搭建好了 Flutter 开发环境。

## 初始化项目

创建项目：查看 -> 命令面板 -> Flutter: New Project，然后等待项目初始化完成。

启动模拟器：`open -a Simulator` (iOS)

运行创建好的 Flutter 项目：`flutter run`
